# 🕌 Serambi Berkah - Sistem Informasi Masjid

Sistem Informasi Masjid modern dengan fitur lengkap untuk management masjid.

## ✨ Fitur Utama

- 🌐 **Website Public** - Informasi masjid untuk jamaah
- 👨‍💼 **Admin Panel** - Management konten mudah  
- 🕋 **Jadwal Sholat** - Manual & otomatis
- 📊 **Laporan Keuangan** - Pemasukan & pengeluaran
- 📷 **Galeri Kegiatan** - Upload foto (max 10MB)
- 📢 **Pengumuman** - Info penting untuk jamaah
- 👥 **Kontak Takmir** - Direct WhatsApp integration
- 📱 **Responsive Design** - Optimal di mobile & desktop

## 🚀 Quick Install

1. **Download & Extract** ke folder web server
2. **Buka** `http://domain-anda/serambi/setup.php`
3. **Ikuti Wizard** - Input database settings
4. **Selesai!** Akses website dan admin panel

## 🔐 Login Admin
- URL: `http://domain-anda/serambi/admin/login.php`
- Default: `admin` / `admin123`

## 📋 Requirements
- PHP 7.4+ 
- MySQL 5.7+
- Apache/Nginx
